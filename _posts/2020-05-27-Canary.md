## # 泄露 canary

- fork 爆破
- 覆盖 canary 的最低位后将其打印。

## # 接收泄露的 canary

- 64 位程序

```python
canary = u64(io.recvuntil('\x30')[-8:-1].rjust(8, '\x00'))
log.success('canary -> ' + hex(canary))
```

## # 调试查看 canary

在程序中对 canary 进行检查的代码处下断点，当程序运行到此处时便可从对应的寄存器中看到 canary 的值。
