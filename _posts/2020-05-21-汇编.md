#### leave
leave 指令在 32 位汇编下相当于：
```
mov esp, ebp 
pop ebp
```
leave 指令将 esp 指向 ebp 所指向的内存，以释放分配给该过程的所有堆栈空间，再将栈顶弹入 EBP 寄存器中。
#### ret
ret 指令在 32 位汇编下相当于：
```
pop eip
```
ret 指令将栈顶弹入 EIP 寄存器中。
#### call
call 指令在 32 位汇编下相当于：
```
push eip + 4
push ebp
mov ebp, esp
```
call 指令依次将下一条指令所在的地址和 EBP 寄存器中的值压入栈内，再将 ebp 指向 esp 所指向的内存。
